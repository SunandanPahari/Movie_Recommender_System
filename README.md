<h1>📽️ Movie Recommender System</h1>
<br>
This project is a content-based movie recommender system built using Python. It suggests similar movies based on the one selected by the user. It uses machine learning and data science techniques to compute movie similarity based on metadata.
<br>
<h2>🧠 Features</h2>
-> Recommend top 5 similar movies<br>
-> Uses content-based filtering (e.g., genres, keywords, cast, crew)<br>
-> Displays movie posters using the TMDb API<br>
-> Built using Streamlit for interactive UI
